    Help & Guide

    Quick answers and usage tips

    How do I add a new action item? +

    Click Add New Item in the sidebar, or press N. Fill in Meeting Date, Action Item, and Owner (required), then click Save. The item appears instantly in All Action Items and the Dashboard.

    How do I edit an existing item? +

    Go to All Action Items, find the row, and click the Edit button. A modal opens with all fields pre-filled. Update any field and click Save Changes.

    How does the Calendar View work? +

    The Calendar plots items by their due date. Navigate months with Prev / Next. Dots are colour-coded: red = High priority, yellow = Medium, green = Low / Completed. Today's date has a blue border.

    How do I sort or filter the table? +

    Click any column header marked with ↕ to sort ascending / descending. Use the filter dropdowns above the table to narrow by Status, Priority, or Owner. The search box matches on title and owner name simultaneously.

    How do I export or back up my data? +

    Go to Export / Import and choose CSV (for Excel / Sheets) or JSON (for developers). Your file downloads immediately. To import, drag-and-drop a file into the import zone — CSV columns must match the listed order.

    Where is my data stored? +

    All data lives in your browser's localStorage. It persists between reloads on the same browser/device. Use Export regularly as a backup. The Reset button in Export / Import restores the original sample data.
